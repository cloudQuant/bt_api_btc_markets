# BTC Markets exchange plugin for bt_api framework.

## Installation

```bash
pip install bt_api_btc_markets
```

## Usage

```python
from bt_api_btc_markets import BtcMarketsExchangeDataSpot, BtcMarketsRequestDataSpot
```

## Features

- REST API support
- HMAC-SHA512 authentication with Base64-encoded secret
- Symbol format: `BTC-AUD`
- Base URL: `https://api.btcmarkets.net`
